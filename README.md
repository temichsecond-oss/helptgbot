# 🎮 Brawl Stars Club Bot — Помічник клубу

Telegram бот для управління клубом Brawl Stars.

---

## ✨ Можливості

| Функція | Опис |
|---|---|
| 📊 Опитування | Адміни створюють власні опитування вручну |
| ⏰ Авто-опитування | Щодня о 13:00 автоматично надсилається опитування у групу |
| 🎯 Цілі клубу | Додавання, перегляд та виконання цілей |
| 💰 Мегакопілка | Оголошення початку/кінця мегакопілки з кнопкою участі |

---

## 🚀 Встановлення

### 1. Вимоги
- Python 3.10+

### 2. Клонування / розпакування
```bash
cd brawl_bot
```

### 3. Встановлення залежностей
```bash
pip install -r requirements.txt
```

### 4. Налаштування `.env`
```bash
cp .env.example .env
```
Відкрийте `.env` та заповніть:

```env
BOT_TOKEN=ВАШ_ТОКЕН          # від @BotFather
GROUP_CHAT_ID=-1001234567890  # ID вашої групи
ADMIN_IDS=123456789           # ваш Telegram ID
```

**Як дізнатись GROUP_CHAT_ID:**
1. Додайте [@username_to_id_bot](https://t.me/username_to_id_bot) у вашу групу
2. Він покаже ID групи (від'ємне число)

**Як дізнатись свій ID:**
- Напишіть [@userinfobot](https://t.me/userinfobot)

### 5. Запуск
```bash
python bot.py
```

## 📋 Команди

### Для всіх учасників групи
| Команда | Дія |
|---|---|
| `/start` або `/help` | Список команд |
| `/goals` | Переглянути цілі клубу |
| `/megastatus` | Статус мегакопілки |
| `/megahistory` | Історія мегакопілок |

### Тільки для адміністраторів 🔒
| Команда | Дія |
|---|---|
| `/newpoll` | Створити нове опитування |
| `/addgoal` | Додати нову ціль клубу |
| `/completegoal` | Позначити ціль виконаною |
| `/cleargoals` | Видалити виконані цілі |
| `/megastart [назва]` | Почати мегакопілку |
| `/megaend` | Завершити мегакопілку |
| `/adminlist` | Список адмінів бота |

## ⚙️ Структура проєкту

```
brawl_bot/
├── bot.py                  # Головний файл запуску
├── config.py               # Конфігурація (читає .env)
├── requirements.txt        # Залежності
├── .env.example            # Шаблон налаштувань
├── handlers/
│   ├── polls.py            # Опитування
│   ├── goals.py            # Цілі клубу
│   ├── megakopilka.py      # Мегакопілка
│   └── admin.py            # Адмін + /help
├── utils/
│   └── scheduler.py        # Планувальник авто-опитування
└── data/                   # Зберігання даних (створюється автоматично)
    ├── goals.json
    └── megakopilka.json
```

## 🔧 Зміна часу авто-опитування

У файлі `config.py` змініть:
```python
DAILY_POLL_HOUR = 13    # година (0-23)
DAILY_POLL_MINUTE = 0   # хвилина (0-59)
```

Часовий пояс: **Europe/Kiev** (Київ)

---

## 📝 Зміна тексту авто-опитування

У файлі `utils/scheduler.py` знайдіть:
```python
DAILY_POLL_QUESTION = "🎮 Як пройшов ваш Brawl Stars сьогодні?"
DAILY_POLL_OPTIONS = [...]
```
та змініть на потрібні.
